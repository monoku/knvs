KNVS

KNVS is a light weight javascript library for canvas animations, it takes a canvas element and creates a continuous animation board. Every element that you draw via the KNVS, is going to be redrawn in the canvas, so any change you do on the element, is gonna be reflected on the canvas.

Here you will find examples of the use of KNVS. You can fork this at: http://github.com/monoku/knvs/

Creating a circle

knvs = Knvs.create('first');

var circle = knvs.draw('circle', {left:150, top: 75, radius: 50, color: 'rgba(218,43,58,1)'});

Moving the circle

knvs = Knvs.create('second');

var circle = knvs.draw('circle', {left:50, top: 75, radius: 50, color: 'rgba(218,43,58,1)'});

circle.morph({left: 250});

Moving the circle with easing

knvs = Knvs.create('third');

var circle = knvs.draw('circle', {left:50, top: 50, radius: 50, color: 'rgba(218,43,58,1)'});

circle.morph({left: 250, top: 200},{transitions:{top: bounce}, duration: 600});

Multiple stuff

knvs = Knvs.create('fourth');

var circle = knvs.draw('circle', {left:50, top: 50, radius: 50, color: 'rgba(218,43,58,1)'});

circle.morph({left: 250, top: 200, color: 'rgba(252,125,73,1)'}, {transitions:{top: bounce, left: easeInOutQuad}, duration: 600});
